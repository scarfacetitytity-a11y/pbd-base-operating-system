# app_build/app_vision.md

## Vision

Build an app that becomes the command centre for:

1. Trading bot
2. Backtesting
3. Trading journal
4. PBD clothing brand planning
5. Prompt library
6. Personal operating system
7. Fitness/PT study support

## First App Version

The first version should be simple:

Pages:
- Dashboard
- Trading Journal
- Backtest Results
- Strategy Library
- Prompt Library
- PBD Brand Ideas
- Study Notes
- Tasks

## Priority

Start with backend organisation before flashy UI.

Build:
1. Clean folder structure
2. Data models
3. Journal storage
4. Backtest storage
5. Prompt storage
6. Basic dashboard

## Rule

Do not overcomplicate the first version.
Make it work first.
Make it beautiful later.
